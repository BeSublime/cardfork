Card Fork

A set of conversation starter cards about AI, crypto, and technology ethics, inspired by the “Hard Questions” segment of the Hard Fork podcast.

Instructions

Shuffle the prompt cards and topic cards separately, and form two decks. Turn over one of each, and use the resulting combination as a conversation starter. If the prompt and topic don't make sense, simply turn over another prompt card. When you reach the end of the prompt deck, re-shuffle it and continue. Spice things up by flipping multiple prompt cards, either at the same time or after some discussion.

Prompt Cards

Would you…
Is it ethical to…
Would you let your kids…
Would you judge someone if they…
When will it seem normal to…
